What is FWI?

The Fire Weather Index (FWI) is a numerical rating of the potential for wildfire ignition and spread. It ranges from low to high risk:

  • Low: FWI < 10 (Green)
  • Moderate: 10 ≤ FWI ≤ 20 (Yellow)
  • High: FWI > 20 (Red)

About the Model

We trained a Ridge Regression model using historical Algerian forest fire data. Features are scaled using StandardScaler before making predictions. Ridge regression helps reduce overfitting and provides stable predictions.

Tips & Applications
